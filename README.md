# Animated Status for BetterDiscord

Модифицированная и актуальная версия плагина **AnimatedStatus** для BetterDiscord. Оригинальный код адаптирован под изменения во внутренней архитектуре Discord и API BetterDiscord. Исправлены критические ошибки авторизации и оптимизирован интерфейс настроек.

## 📌 Описание изменений и предыстория

Оригинальный плагин (авторы: **toluschr** и **SirSlender**) перестал функционировать из-за обновлений механизмов кеширования модулей Webpack в Discord, что приводило к невозможности получения токена авторизации и ошибкам при запуске.

**Что было сделано в рамках данного форка:**
* **Исправление авторизации:** Полностью переписан метод перехвата токена. Теперь плагин корректно и безопасно запрашивает токен через актуальный контейнер `AuthenticationStore` самого BetterDiscord, исключая сбои при загрузке.

## ⏱ Настройка задержки времени (Таймаут кадра)

В конфигурации плагина минимально возможный порог задержки жестко ограничен значением **2900 мс** (2.9 секунды) для предотвращения избыточной нагрузки на серверы. При установке таймингов необходимо учитывать специфику лимитов Discord (Rate Limits):

* **Интервал 2900 – 3000 мс:** Экстремально низкое значение. Статус обновляется максимально быстро, однако существует высокий риск получить сетевую ошибку `429 Too Many Requests`. Это означает, что серверы Discord временно заблокировали отправку запросов, посчитав активность спамом. В таком случае задержку необходимо увеличить.
* **Интервал 5000 – 10000 мс (Рекомендуемый):** Оптимальный диапазон для стабильной работы. Обеспечивает плавную смену кадров анимации и полностью исключает срабатывание защитных триггеров Discord. Плагин может работать непрерывно в режиме 24/7.

> **Техническая рекомендация:** Рекомендуется устанавливать глобальный таймаут на уровне `10000` (10 секунд) либо выставлять индивидуальные задержки для ключевых кадров не менее `5000` (5 секунд).

## 🚀 Установка и использование

1. Поместите файл `Animated_Status.plugin.js` в директорию плагинов BetterDiscord на вашем устройстве (открыть её можно через *Настройки пользователя -> Плагины -> Открыть папку плагинов*).
2. Активируйте плагин в панели BetterDiscord с помощью переключателя.
3. **Обязательно перезагрузите клиент Discord** (сочетание клавиш `Ctrl + R`), чтобы плагин успешно инициализировал токен авторизации при перезапуске модуля.
4. Перейдите в настройки плагина (иконка шестерёнки) для формирования таблицы кадров анимации и нажмите кнопку **Save**.
